JavaScript

A programming language for manipulating web pages.

Operators

symbols for doing comparisons and arithmatic in programming languages. (ie. ===, +=, +, %)

Variables

A variable is an abstract storage location paired with an associated symbolic name, which contains some known or unknown quantity of information referred to as a value.

Variable naming conventions

In JavaScript we use lowerCamelCase to name variables.

Functions

A set of repeatable code.

Methods

Functions that are attached to an object.

Arguments

Arguments are the data you put into a function.

Parameters

Parameters are place holders for a function.

Return

Keyword for returning a value from a function.

Chaining methods

Using methods on methods.

Strings

A specific data type encased in quotation marks.

Booleans

True and false or binary output data type.

Undefined

Data type the signify no data.

NaN

Not a number data type.

Null

When a variable doesn't point to something.

Alert

A javascript function that pushes a message to the user.

Comments

Messages in code that are not run.

JQuery

A javascript library.

Attributes

Element indicators aka attributes such as id class type src.
